== Homepage Installation ==

Step-1. Go to Dashboard -- Pages -- Add new page -- Add page attribute (Home Page) -- Publish it.
Step-2. Go to Dashboard -- Settings -- Reading -- Click on static page -- Select home page -- Save changes.

Layout section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Site Layout Section
 -- Check to Box Layout for box width layout.  -- Publish it.

Slider section
Step-1. Go to Dashboard -- Post -- Add new Category -- Add new post -- Add title, content, select category and featured image -- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Slider Section
 -- Add Section Title also Select the page -- Publish it.

Services section
Step-1. Go to Dashboard -- Page -- Add new Page -- Add title, content and featured image -- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Services Section
 -- Add Section Title also Select the page -- Publish it.

Footer section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Footer Section -- Add copyright text -- Publish it.
